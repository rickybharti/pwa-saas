# Tiered Product Scope

## Tier 1 — Must Have (MVP)

### 1) One-Tap Install PWA
- Guided install UI with platform-specific steps
- Creator-branded icon/splash/theme
- Performance budget: LCP < 1.5s on mid-tier mobile
- Offline shell and app-like nav

### 2) Web Push Notification Engine
- Push subscriber capture after first value action
- Broadcast campaigns and scheduled sends
- Segments: free, paid, topic-based
- Drip automations triggered by signup/purchase
- Message A/B test support
- Metrics: sent, delivered, open, CTR, conversion

### 3) Instant Content Delivery System
- Role and entitlement-gated content
- Time-window unlock rules
- One-time secure links for high-value prompt packs
- Downloadable assets with signed URLs
- Video embeds with per-asset tracking

### 4) Monetization Layer
- Stripe one-time checkout
- Stripe subscriptions (monthly/yearly)
- Coupon support and post-purchase upsells
- Lightweight revenue dashboard
- UPI support for India creator monetization (intent/QR/collect)

### 5) Creator Dashboard
- Total subscribers and active push subscribers
- Revenue and conversion rate snapshots
- Retention graph (cohort-level)
- Click heatmap by content block/link slot

## Tier 2 — Growth Features

### 6) Funnel Builder
- Landing page builder (template-driven blocks)
- Bio-link mode for social profiles
- UTM + source tracking by campaign/reel

### 7) Lead Segmentation
- Multi-dimensional segments: niche, stage, paid status, behavior
- Dynamic rules + static tags
- Segment-level campaign targeting

### 8) Automated Sequences
- Day-based and event-based sequences
- Skip/branch logic based on click/purchase behavior
- Sequence health metrics and drop-off analysis

### 9) Analytics Intelligence
- Best send-time recommendations
- Best-converting content insights
- Funnel drop-off by step
- Revenue per subscriber (RPS)

## Tier 3 — Differentiators (Moat)

### 11) Community Lite
- Comments per post/content item
- Reactions and lightweight polls
- Feedback prompts tied to retention scoring

### 13) Referral Engine
- Invite links + attribution
- Reward types: unlock content, discount, bonus content
- Fraud controls: IP/device velocity checks

## White-Label + SaaS Packaging
- Custom domains (`creatorname.app`)
- Full creator branding (logo, colors, splash, no watermark by plan)
- Plan-based feature flags and usage limits
